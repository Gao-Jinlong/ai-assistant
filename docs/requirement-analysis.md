# 需求分析文档

## 项目信息

项目名称：人生驾驶舱
项目描述：人生驾驶舱是一个基于 AI 的个人教练，用于帮助用户发掘自身的成长潜力，制定关键目标和阶段规划，帮助用户观察计划执行进度，发掘成长过程中的痛点难点，提供针对性建议和动态调整。

### 主要用户角色

1. 希望获得职业发展建议的用户
2. 希望改善自身性格缺陷，通过刻意练习提升沟通能力的用户
3. 希望提升自身身体素质的健身爱好者

### 典型用户故事
1. **小李 - 应届毕业生**
   > "作为一名应届毕业生，我不确定自己适合什么样的工作。我想了解自己的性格特点和优势，找到匹配的职业方向。"

2. **小张 - 因为内向不善于表达，经常对他人或自己造成不必要的困扰**
   > "我因为内向不善于表达，经常对他人或自己造成不必要的困扰。我想改善自己的性格缺陷，提升沟通能力。"

3. **小王 - 希望提升身体素质的健身爱好者**
   > "我是一名健身爱好者，希望提升自己的身体素质，保持健康的生活方式。"

## 产品核心功能优先级

- 驾驶舱
- 智能测评系统
- 提升路径规划 Agent
- 规划执行陪伴 Agent
- 成长档案 Agent

## 详细需求规格

### 驾驶舱
- 驾驶舱包含至少3个模块：职业规划、性格提升、身体管理
- 每个模块包含至少3个功能：测评、规划、陪伴
- 测评、规划、陪伴模块之间可以自由跳转

### 智能测评系统
- AI快速测评通过10个问题基本判断用户基本诉求和基本情况
- 支持至少3种标准化测评（DISC、MBTI、霍兰德）
- 
- 测评结果保存并支持历史查看

### 职业路径规划引擎
- 基于用户画像提供至少3条职业发展路径
- 每条路径包含时间轴和里程碑设计
- 提供针对性的技能提升建议
- 支持用户自定义职业目标
- 提供行业洞察和趋势分析

### AI教练陪伴系统
- 支持自然语言对话
- 根据用户目标提供定制化建议
- 提供情绪疏导和心理支持
- 支持用户自定义目标
- 提供行业洞察和趋势分析
